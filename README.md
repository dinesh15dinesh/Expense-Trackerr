# Advanced Expense Tracker using Python
## Features

* Add new expenses with date, category, description, and amount
* View all recorded expenses
* Calculate total expenses
* Generate category-wise expense summaries
* Generate monthly expense reports
* Delete unwanted expense records
* CSV-based persistent data storage
* Automatic loading of saved expenses

## Technologies Used

* Python 3
* CSV File Handling
* Dictionaries and Lists
* Exception Handling
* Modular Programming

## How to Run

1. Install Python 3 on your system.
2. Clone the repository:

git clone https://github.com/dinesh15dinesh/Expense-Tracker.git

3. Navigate to the project directory:

cd Expense-Tracker

4. Run the application:

python expense_tracker.py
